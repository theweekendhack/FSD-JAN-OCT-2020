Selectors

Selectors can be a word and/or a pattern used to select an element element(s) from your HTML document

  • Element (or Tag) selector
  • ID selector
  • Class
  • Universal
  • Attribute
  • Pseudo-class
  • Pseudo-element
  • Combinators
    1. Child Combinator (a > b)
    2. Descendant selector(a b)
    3. Adjacent sibling selector (a + b)
    4. General sibling selector (a ~ b)
